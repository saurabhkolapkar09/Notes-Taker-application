# Notes-Taker-application

This is a maven web application created with the help of Eclipse IDE.
With the help of application user can mentain the notes , which will stored in database. 

## Customer
    * Add notes
    * Update notes
    * Delete notes
 
## Technologies Used
    * Java
    * Hibernate
    * Servlets and JSP
    * MySql Database
    * Tomcat Server
